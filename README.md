# butane-jekyll-theme

## Dependancies
- Ruby 2.2
- Jekyll 3.0

## Getting Started
- `git clone https://github.com/alexcarpenter/butane-jekyll-theme.git`
- `cd butane-jekyll-theme`
- `jekyll serve`

## Demo
To see a working demo of the Butane Jekll Theme visit [alexcarpenter.me](http://alexcarpenter.me).
